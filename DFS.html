Depth-First Search

void DFS(int v) {

visited[v] = true;
cout<< v<< " " ;

for (i = adj[v].begin(); i != adj[v].end(); ++i)

if (!visited[*i])
DFS(*i);

}

Time Complexity

Adjacency list implementation:
Time Complexity: O(v+e)

Adjacency matrix implementation:
Time Complexity: O(v^2)

where v stands for vertices and e stands for edges
