Array Implementation

In array implementation, the queue is formed by using the array. Front and rear variables point to the position from where insertions and deletions are performed in a queue.

Linkedlist Implementation

In the linked queue, 2 pointers are maintained in the memory(front & rear pointer). The front pointer contains the address of the starting element of the queue while the rear pointer contains the address of the last element of the queue.
