Max Heap

max_heap:
Begin
  Declare function max_heap()
   Declare j, t of the integer datatype.
  Initialize t = a[m].
  j = 2 * m;
  while (j<=n) do
   if (j< n && a[j+1] >a[j]) then
     j = j + 1
   if (t > a[j]) then
     break
   else if (t <= a[j]) then
     a[j / 2] = a[j]
     j = 2 * j
   a[j/2] = t
  return
End.
build_maxheap:
Begin
 Declare function build_maxheap(int *a,int n).
  Declare k of the integer datatype.
  for(k = n/2; k >= 1; k--)
   Call function max_heap(a,k,n)
End.
