Min Heap

In a Min-Heap the key present at the root node must be less than or equal among the keys present at all of its children. The minimum key element is present at the root.

Max Heap

In a Max-Heap the key present at the root node must be greater than or equal among the keys present at all of its children. The maximum key element is present at the root.

Heap Sort

Heap sort processes the elements by creating the min heap or max heap using the elements of the given array. At each step, the root element of the heap gets deleted and stored into the sorted array and the heap will again be heapified.
